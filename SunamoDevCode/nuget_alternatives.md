# NuGet Alternatives to SunamoDevCode

This document lists popular NuGet packages that provide similar functionality to SunamoDevCode.

## Overview

Code generation and development utilities

## Alternative Packages

### Microsoft.CodeAnalysis
- **NuGet**: Microsoft.CodeAnalysis.CSharp
- **Purpose**: Roslyn compiler API
- **Key Features**: Syntax trees, semantic analysis, code generation

### CodeGeneration.Roslyn
- **NuGet**: CodeGeneration.Roslyn
- **Purpose**: Source generators
- **Key Features**: Compile-time code generation

### T4
- **NuGet**: Microsoft.VisualStudio.TextTemplating
- **Purpose**: Text template transformation
- **Key Features**: Code generation from templates

### Scriban
- **NuGet**: Scriban
- **Purpose**: Template engine
- **Key Features**: Fast, safe text generation

## Comparison Notes

Roslyn for C# code analysis. Source generators for compile-time generation. Scriban for text templates.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **Microsoft.CodeAnalysis**: Roslyn compiler API
- **CodeGeneration.Roslyn**: Source generators
- **T4**: Text template transformation
